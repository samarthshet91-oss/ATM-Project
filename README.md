# 💳 ATM Machine Project

A simple ATM simulation project built using Python (Tkinter) and C backend.

## 🚀 Features
- 🔐 Login system (Account + PIN)
- 💰 Check Balance
- 💸 Deposit Money
- 🏧 Withdraw Money
- 📊 Transaction History
- 🔄 Smooth UI navigation
